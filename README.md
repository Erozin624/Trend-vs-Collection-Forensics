## 📖 What Is Trend-vs-Collection-Forensics?

Trend-vs-Collection-Forensics is an application designed to help you understand why many Key Performance Indicator (KPI) trends might not be what they seem. Instead of taking trends at face value, this tool helps identify common issues like measurement errors or changes in data collection that can create false trends.

This project covers several common causes of incorrect trends:

- Schema drift: how changes in the data structure affect measures
- Coverage loss: missing data that skews results
- Time shifts: delays or shifts in data recording times
- Backfills: corrections or additions of historical data
- Dedupe: removing duplicate records
- Sampling effects: partial data affecting the trend
- Mix change: changes in segments or groups within the data

It also offers protocols for reviewing evidence carefully (Trend Courtroom), ways to test parts of your data for consistency (segment invariance tests), and ready-to-use checklists and a scoring system (Trend Report Card) to help you make safer decisions.
